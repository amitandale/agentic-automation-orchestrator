# DM Outreach — Direct Message Skill

## Prerequisites
- Load `SOUL-dm.md` for assistant persona (NEVER use author persona for DMs).
- Confirm target has been scored by Hermes (minimum score threshold: 6/10).
- Confirm target has not been previously contacted (check Paperclip audit trail).
- Confirm inbox-triage plugin has no opt-out flag for this target.

## Step-by-Step Instructions

### 1. Prepare Message
- Send the influencer's profile data to Hermes for drafting.
- Hermes drafts a personalized first-contact message using the assistant persona.
- Message must include:
  - A specific reference to the influencer's recent content (shows genuine interest).
  - A brief mention of why their audience would enjoy the reading material.
  - A tracked free reading link (generated by link-generator plugin).
  - A clear but non-pushy call to action.

### 2. Validate Message
- Is the message under 500 characters? (Platform DM limits vary.)
- Does it use assistant persona, NOT author persona?
- Does it contain a valid tracked link from the link-generator plugin?
- Is the tone warm, professional, and non-spammy?

### 3. Send DM
- Navigate to the influencer's profile on the relevant platform.
- Open the DM/message interface.
- Type the message with natural delay.
- Send.

### 4. Log and Schedule Follow-Up
- Log the DM content, target handle, platform, and timestamp to Paperclip.
- Schedule a follow-up task in Paperclip for 7 days later.
- If no response after 7 days: send ONE follow-up message (shorter, lighter).
- If no response after 14 days: archive the outreach attempt. Do not contact again.

### 5. Response Handling
- If the influencer responds positively: escalate to operator via WhatsApp.
- If the influencer responds negatively or asks to opt out: mark in inbox-triage, cancel all follow-ups.
- If the influencer asks questions: route to Hermes for a follow-up draft.

## Critical Rules
- Maximum 3 new DMs per day across all platforms.
- Never send DMs between 10 PM and 8 AM in the recipient's timezone.
- Never send the same message template to two different influencers.
