# DM Outreach — Direct Message Skill

## Prerequisites
- Read the task's `funnel` parameter (`religious` or `secular`).
- Load `SOUL-dm.md` for assistant persona (NEVER use author persona for DMs).
- Confirm target has been scored by Hermes (minimum score threshold: 6/10).
- Confirm target has not been previously contacted (check Paperclip audit trail).
- Confirm target has no opt-out flag (check Paperclip audit trail for opt-outs).

## Step-by-Step Instructions

### 1. Detect Language
- Read the `language` attribute from the influencer's profile data (captured by the `influencer-scout` skill).
- This is the language the DM must be drafted in.

### 2. Prepare Message
- Send the influencer's profile data, **funnel**, and **target language** to Hermes for drafting.
- Hermes drafts a personalized first-contact message using the assistant persona.
- Hermes queries Qdrant filtered by `funnel` AND `language` for relevant source material.
- Message must include:
  - A specific reference to the influencer's recent content (shows genuine interest).
  - A brief mention of why their audience would enjoy the reading material.
    - If `funnel = religious`: position the trilogy as deep Torah study / spiritual insight.
    - If `funnel = secular`: position Book 4 as consciousness / self-improvement material.
  - A tracked free reading link (generated by link-generator skill via OpenClaw).
  - A clear but non-pushy call to action.
- **The message must be drafted natively in the detected language** — never translate.

### 3. Validate Message
- Is the message under 500 characters? (Platform DM limits vary.)
- Does it use assistant persona, NOT author persona?
- Does it contain a valid tracked link?
- Is the tone warm, professional, and non-spammy?
- Is it written in the correct language for this influencer?
- Does it reference the correct funnel's material (not cross-funnel)?

### 4. Send DM
- Navigate to the influencer's profile on the relevant platform.
- Open the DM/message interface.
- Type the message with natural delay.
- Send.

### 5. Log and Schedule Follow-Up
- Log the DM content, target handle, platform, language, funnel, and timestamp to Paperclip.
- Schedule a follow-up task in Paperclip for 7 days later (same funnel + language).
- If no response after 7 days: send ONE follow-up message (shorter, lighter, same language).
- If no response after 14 days: archive the outreach attempt. Do not contact again.

### 6. Response Handling
- If the influencer responds positively: escalate to operator via WhatsApp.
- If the influencer responds negatively or asks to opt out: mark opt-out in Paperclip audit trail, cancel all follow-ups.
- If the influencer asks questions: route to Hermes for a follow-up draft (same funnel + language).

## Critical Rules
- Maximum 3 new DMs per day across all platforms.
- Never send DMs between 10 PM and 8 AM in the recipient's timezone.
- Never send the same message template to two different influencers.
- Never cross funnels — a religious influencer gets the trilogy pitch, a secular one gets the Book 4 pitch.
